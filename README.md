# Unity Elevator System 

##  Project Overview

This project simulates an elevator system built in Unity. It handles floor selection, movement logic, and user interaction.

##  Features

* Elevator movement between floors
* Button-based floor selection
* Smooth transitions
* Basic UI

##  Approach

* Used C# scripts for elevator logic
* Implemented state-based movement system
* Optimized assets for performance

##  How to Run

1. Open project in Unity
2. Open main scene
3. Press Play

##  Submission 

* Unity Package included separately
